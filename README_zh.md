项目说明
纯粹出于兴趣，我将编程作为一种爱好，其性质类似于玩CS和搭建乐高。我想尝试最好能有专业从业人员或者大手子的关注，获得指导，甚至帮我给这个项目来点issues。我个人没有理工科或计算机科学的教育背景，本科是学油画出身的，在数学或程序员常遇到的实际应用场景方面基础较为薄弱，因此代码中可能存在很多问题。
强制要求
必须使用 MSVC 编译器​ 并支持 C++20 标准。
目标平台为 Windows，因为代码大量使用了 Windows 平台的独有 API​ 以及 C++20 的语法特性。

在 simd_floating.hpp中，我使用本库的其他部分编写了许多数学函数，这些函数可以作为使用本库的示例。考虑到 simd_def.hpp中定义的两种主要向量类型仍在完善过程中，这些数学函数的实现代码未来可以变得更加优雅。

待办事项（按优先级排序）：
  1. 改进浮点运算相关函数​ <- 目前正在做这部分工作
  2. 添加对复数的支持
  3. 添加对 AVX512​ 指令集的支持
  4. 添加对 ARM​ 架构的支持
  5. 移除对 Windows 平台 MSVC 编译器独有 API 的依赖​ <- 我认为这一点非常重要
  6. 优化现有由复合指令实现的函数的性能
